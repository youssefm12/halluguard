# Contributing to HalluGuard AI

First off, thank you for considering contributing to HalluGuard AI! It's people like you that make HalluGuard AI such a great tool.

## Where do I go from here?

Please read our `docs/architecture.md` to get an understanding of the system's layout.

## Bug Reports

Submit bug reports using the GitHub issue tracker. Please include:
- A quick summary and/or background
- Steps to reproduce
- What you expected would happen
- What actually happens
- Notes (possibly including why you think this might be happening, or stuff you tried that didn't work)

## Pull Requests

1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes.
4. Make sure your code lints (run `make lint`).
5. Issue that pull request!

Happy coding!
